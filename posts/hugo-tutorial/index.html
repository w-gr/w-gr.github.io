Hugo - tutorial

March 1, 2019

Hugo is a static site generator written in Go. This tutorial is based on these: install, quick start, integrate with github

Install

sudo apt-get install hugo
# Check if have hugo
hugo version

Create a new site

# this will create a new hugo site in a folder named blogs
hugo new site blogs

Add a theme

cd blogs
# Download the theme
git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
# Edit your config.toml configuration file
# and add the Ananke theme.
echo 'theme = "ananke"' >> config.toml

Add one page

hugo new posts/hugo-tutorial.md

Start the hugo server

hugo server
# now you can visit http://localhost:1313/ to see your site.

Integrate with github pages

Please make sure the github account is available.

git submodule add master git@github.com:<USERNAME>/<USERNAME>.github.io.git public

Add the deploy shell sript

Add this shell script file with a name like deploy.sh to the site root folder.

#!/bin/bash

echo -e "\033[0;32mDeploying updates to GitHub...\033[0m"

# Build the project.
hugo # if using a theme, replace with `hugo -t <YOURTHEME>`

# Go To Public folder
cd public
# Add changes to git.
git add .

# Commit changes.
msg="rebuilding site `date`"
if [ $# -eq 1 ]
  then msg="$1"
fi
git commit -m "$msg"

# Push source and build repos.
git push origin master

# Come Back up to the Project Root
cd ..

Deploy to github

When every thing is OK, we can run the deploy.sh to push it to github.

./deploy.sh 'initial blog'

That’s all.
