Laravel Event List

There doesn’t have a convenient way to get the full list of Laravel system defined events. So we have to search the source code to get the list. Laravel supports two ways to create an event: 1. Event class 2. Closure based event with a given name

The event classes

The event classes are typically stored in Events directory

# The events directories
find vendor/laravel/framework/ -name 'Events' -type d | grep -v '/Illuminate/Events' | sort
# The event classes, exclude some base classes, traits, etc.
find vendor/laravel/framework/ -path '*/Events/*' -type f | grep -vE '/Illuminate/Events|ConnectionEvent|CacheEvent|Dispatcher|Dispatchable|ConnectionEvent' | sort

The closure based events

The are registed by the function listen and fired by dispatch, so we can search the usages of listen and dispatch functions in the source code.

# Find the listen usages
grep -r ">listen(" 'vendor/laravel/framework/'
# Find the dispatch usages
grep -r ">dispatch(" 'vendor/laravel/framework/'
